# GCloud-Go

This repo contains a Golang implementation of the [Google Cloud SDK CLI](https://docs.cloud.google.com/sdk/docs/install-sdk), which provides the CLI `gcloud`.

# Install

```sh
export GOPATH="$HOME/go"
go install github.com/flyingobsidian/gcloud-go
"$GOPATH/bin/gcloud-go" --help
```
